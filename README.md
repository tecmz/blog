# ModStartBlog

魔众博客系统是一个小巧强大的博客程序，后台可以高扩展，拥有手机端、电脑端自适应。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
